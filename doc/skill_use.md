## 二、详细调用链流程

### 阶段 1: 用户输入提交

```
┌────────────────────────────────────────────────────────────────────────┐
│  用户操作: 在 PromptInput 输入 "/skill-name arg1 arg2" 并回车            │
└────────────────────────────────────────────────────────────────────────┘
                          │
                          ▼
┌────────────────────────────────────────────────────────────────────────┐
│  REPL.tsx: onSubmit (useCallback)                                      │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   │
│  1. repinScroll() - 滚动到底部                                          │
│  2. 检查立即命令 (immediate command)                                     │
│     └── 匹配命令的 `immediate: true` 属性                                │
│  3. 检查空闲返回对话框 (idle-return dialog)                              │
│  4. 添加到历史记录 (addToHistory)                                        │
│  5. 恢复暂存的提示词 (stash restoration)                                 │
│  6. 清除输入并显示占位符 (setUserInputOnProcessing)                      │
│  7. 调用 handlePromptSubmit                                             │
└────────────────────────────────────────────────────────────────────────┘
                          │
                          ▼
```

- `loadSkillsFromCommandsDir()`: 从 `/commands/` 目录加载（遗留）
- `createSkillCommand()`: 创建技能命令对象

- 流式响应处理
